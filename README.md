# Tic-Tac-Toe

**Prerequisites :**

- [G++](https://gcc.gnu.org/)  
- [make](https://www.gnu.org/software/make/)  

***

**Compile and Run :**

``$ make run``  
or  
``$ g++ src/*.cpp -std=c++2a -o tic_tac_toe``  
``$ ./tic_tac_toe``
