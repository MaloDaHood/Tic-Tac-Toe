# Tic-Tac-Toe

**Prerequisites :**

- <a href="https://gcc.gnu.org/">G++</a>
- <a href="https://www.gnu.org/software/make/">make</a> (optional)

***

**Compile and Run :**

``$ make run`` <br />
or <br />
``$ g++ src/*.cpp -std=c++2a -o tic_tac_toe`` <br />
``$ ./tic_tac_toe``
