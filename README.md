# Tic-Tac-Toe

This project is a training to OOP and to basic AI implementation.  

## **Prerequisites :**

- [G++](https://gcc.gnu.org/)  
- [make](https://www.gnu.org/software/make/) (optional)

***

## **Compile and Run :**

- ### On Linux / UNIX

``$ make run``  
or  
``$ g++ src/*.cpp -std=c++2a -o tic_tac_toe``  
``$ ./tic_tac_toe``

- ### On Windows

``$ g++ src/*.cpp -std=c++2a -o tic_tac_toe.exe``  
``$ ./tic_tac_toe.exe``
